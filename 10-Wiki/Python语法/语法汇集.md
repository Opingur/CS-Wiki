# 1. 输入输出

## 输出

```python
print("Hello World")
```

等价于：

```cpp
cout << "Hello World" << endl;
```

>print()里面加"end = xxx"可以指定输出末尾。输出末尾默认是换行符
```
print("Hello", end=" ")   # 打印 "Hello" 后，打印一个空格（不换行）
print("World")            # 打印 "World" 后，默认打印换行符
```
## `end` 只影响它所在的那一次 `print()`
```

print("A", end="--")   # A 后面跟 --
print("B", end="")     # B 后面跟 空（不换行）
print("C")             # C 后面跟 换行（默认）

输出：`A--BC\n`

每次 `print()` 的 `end` 都是独立的，不会永久改变后续 `print()` 的行为。
```

---

## 输入

```python
name = input()
```

等价于：

```cpp

```

## 格式化输出


# 14. 引用语义（重点）

Python：



# 21. 与C++最大的思维差异

## C++

默认值语义

```cpp
vector<int> b = a;
```

复制。

---

## Python

默认引用语义

```python
b = a
```

共享对象。

---

## Rust

默认所有权转移

```rust
let b = a;
```

move。

---

因此：

```text
Python对象模型
≈ Java引用模型
≠ C++值语义
≠ Rust所有权模型
```
